
# Milesto 

Milesto is a full-stack project and task management application designed to help students and teams manage academic projects efficiently, especially final-year projects.It provides a centralized dashboard where users can create projects, assign tasks, upload documents, collaborate with team members, and track overall progress.

---

##  Features

- Secure authentication (Signup / Login)
- Create and manage multiple projects
- Add tasks with priority and deadlines
- Track task status (Pending / In Progress / Completed)
- Upload and manage project documents
- Team collaboration with member roles
- Centralized dashboard overview
- AI-based insights for project tracking

---



##  Tech Stack

### Frontend
- React
- Tailwind CSS
- Context API
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### Deployment
- Frontend: Vercel
- Backend: Render
